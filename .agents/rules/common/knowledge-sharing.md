# Knowledge Sharing Rules

## 목적

공용 AI 계정의 핵심 목적은 비용 절감보다 지식 공유화다.

## 기본 루프

1. PR 또는 작업 단위마다 한 줄 노하우를 남긴다.
2. 반복 가능한 프롬프트는 Prompt DB 후보로 기록한다.
3. 좋은 사례와 실패 사례를 weekly 회고/데모데이에서 공유한다.
4. 승인된 프롬프트와 workflow는 agent asset 또는 skill로 승격한다.

## Agent가 도와야 할 일

- 작업 완료 후 재사용 가능한 노하우를 요약한다.
- Prompt DB 후보가 있으면 목적/입력/출력/주의점을 정리한다.
- 출처가 있는 지식은 source attribution rule에 맞춰 기록한다.
