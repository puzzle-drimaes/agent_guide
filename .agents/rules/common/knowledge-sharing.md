# Knowledge Sharing Rules

## 목적

공용 AI 계정의 핵심 목적은 비용 절감보다 지식 공유화다.

## 기본 루프 (1차 도입: 강제 아님, 권장)

기록을 강제하지 않는다. 부담 없는 공유를 우선한다.

1. 공유하고 싶은 스킬/프롬프트 .md는 공유 폴더(Google Drive)에 올린다.
2. 다른 사람이 올린 .md를 선택해 자기 프로젝트(`.agent-packs/externals/`)에 적용한다.
3. 반복해서 쓸 만한 프롬프트만 Prompt DB 후보로 기록한다.
4. 회고/데모데이 공유와 노하우 한 줄 메모는 권장이며 의무가 아니다.
5. 충분히 검증되면 agent asset 또는 skill로 승격한다.

## 예외 없이 지키는 것

- 보안: 금지 데이터(고객 개인정보 / credential / token 등) 미입력.
- 출처: 외부 자료는 source attribution rule에 맞춰 기록.

## Agent가 도와야 할 일

- 사용자가 원하거나 재사용 가치가 분명할 때만 노하우를 한 줄로 요약 제안한다 (강제 아님).
- 공유 폴더에 올릴 .md 후보가 있으면 민감정보 제거를 돕고 정리한다.
- Prompt DB 후보가 있으면 목적/입력/출력/주의점을 정리한다.
- 출처가 있는 지식은 source attribution rule에 맞춰 기록한다.
