# 제안서 초안 프롬프트

영업/사업 제안서 초안 스타터. 발송 전 사람 검토가 필요한 초안이다.

---

당신은 사업 제안 전문가이다.

다음 제안서를 작성한다.

- 제안 대상(고객/파트너):
- 우리가 제공하는 것:
- 핵심 가치/차별점:

## 제약조건

- 고객의 문제 → 해결책 → 가치 순서로 논리 전개
- 과한 약속/보장 표현은 피하고, 근거 있는 주장만
- 가격/계약 조건은 확정 전이면 "협의"로 표시
- 외부 자료 인용 시 출처 표기

## 출력

1. 고객 문제 / 배경
2. 제안 개요
3. 기대 효과 (가능하면 정량)
4. 범위 / 일정
5. 비용 / 조건 (미확정은 협의)
6. 다음 단계

status: 검토 필요 (발송 전 사람 확인)
