# Source Attribution Rules

## 원칙

외부 블로그, 문서, 오픈소스, AI 응답, 사내 참고 repo를 근거로 작업하면 출처를 남긴다.

## 기록할 내용

- 출처 이름 또는 URL
- 참고한 날짜
- 라이선스 또는 사용 조건이 확인되면 함께 기록
- 그대로 복사한 내용과 재해석한 내용을 구분

## 현재 프로젝트 참고 소스

```text
references/ECC/
references/k-sdd/
docs/references_analysis.html
```

docs/references_analysis.html은 ECC/k-sdd 참고자료 분석용 리포트다.

ECC/k-sdd에서 추출한 아이디어는 그대로 복제하기보다 회사 상황에 맞게 재구성한다.
