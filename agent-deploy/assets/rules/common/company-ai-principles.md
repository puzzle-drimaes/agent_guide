# Company AI Principles

## 목적

AI agent는 단순 자동화 도구가 아니라 회사 지식을 축적하고 재사용하게 만드는 업무 인프라다.

## 원칙

1. 같은 프로젝트에서는 같은 agent 규칙을 사용한다.
2. 프로젝트별 규칙은 project scope에 남기고, user/global 설정은 선택으로 둔다.
3. AI가 만든 중요한 판단은 문서, PR, TODO, Prompt DB 후보로 남긴다.
4. 반복 가능한 프롬프트와 workflow는 개인 노하우로 끝내지 않고 자산화한다.
5. Claude/Codex/Gemini/Kiro 등 도구가 달라도 의미상 동일한 규칙을 적용한다.

## 작업 완료 시 남길 것

- 무엇을 변경했는지
- 어떤 규칙을 적용했는지
- 재사용 가능한 노하우가 있는지
- 남은 의사결정이나 위험이 있는지
